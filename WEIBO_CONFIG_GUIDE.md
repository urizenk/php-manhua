# 🔗 微博主页链接配置指南

## 📋 功能说明

现在您可以在后台管理界面灵活配置微博主页跳转链接，无需修改代码或配置文件。

## 🎯 配置步骤

### 1. 登录后台管理

访问：http://8.149.138.212:9090/admin88/

- 账号：`admin`
- 密码：`admin123`

### 2. 进入网站配置

登录后，在控制台页面点击 **"网站配置"** 按钮，或直接访问：
http://8.149.138.212:9090/admin88/site-config

### 3. 配置微博信息

在 **"社交媒体配置"** 区域，您可以设置：

#### 微博主页链接
- 输入您的微博主页完整URL地址
- 例如：`https://weibo.com/u/1234567890`
- 或：`https://weibo.com/your-username`

#### 微博按钮显示文本
- 自定义按钮上显示的文字
- 例如：`微博@资源小站`
- 或：`关注我的微博`

### 4. 实时预览

配置页面提供实时预览功能，您可以：
- 输入链接和文本后立即看到效果
- 确认样式符合预期后再保存

### 5. 保存配置

点击 **"保存配置"** 按钮，配置立即生效！

---

## 🌐 生效范围

配置保存后，以下页面的微博按钮都会自动更新：

1. **首页** - 欢迎卡片中的微博按钮
2. **反馈页面** - 联系方式中的微博链接
3. **备用地址页面** - 社交媒体链接区域

## 💡 使用建议

### 推荐的微博链接格式

```
https://weibo.com/u/[您的用户ID]
```

或

```
https://weibo.com/[您的个性域名]
```

### 推荐的显示文本

- `微博@资源小站`
- `关注我的微博`
- `微博@海の小窝`
- `@资源分享站`

---

## 🔧 技术实现

### 数据存储

配置信息存储在数据库的 `site_config` 表中：

| 配置键 | 说明 | 默认值 |
|--------|------|--------|
| `weibo_url` | 微博主页链接 | `https://weibo.com/` |
| `weibo_text` | 按钮显示文本 | `微博@资源小站` |

### 前台读取

所有前台页面都会从数据库动态读取配置，无需重启服务器或清理缓存。

## ⚠️ 注意事项

1. **链接格式**
   - 必须以 `http://` 或 `https://` 开头
   - 确保链接可以正常访问
   - 建议使用微博的完整URL

2. **显示文本**
   - 建议不超过20个字符
   - 避免使用特殊符号
   - 保持简洁明了

3. **安全性**
   - 只有管理员可以修改配置
   - 所有输入都经过安全过滤
   - 支持CSRF防护
